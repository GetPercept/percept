The Moat

Context Intelligence Layer

Transcription is a commodity. What happens after is what matters. The CIL transforms raw speech into structured, actionable context ‚Äî so "email the client" actually works because your agent knows who the client is.

  • Two-pass entity extraction ‚Äî fast regex + LLM semantic pass
  • Relationship graph with weighted edges and linear decay
  • 5-tier entity resolution ‚Äî exact, fuzzy, contextual, recency, semantic
  • NVIDIA NIM embeddings + LanceDB vector search
  • FTS5 full-text search with porter stemming
  • Context packets ‚Äî single JSON blob with everything an agent needs
  • SQLite persistence ‚Äî single file, zero deps, WAL mode

Percept Protocol

A framework-agnostic JSON schema for voice ‚Üí intent ‚Üí action handoff. Six event types. Three transports. Unix composable.

Your agent framework doesn't matter. LangChain, CrewAI, AutoGen, ü¶û OpenClaw, or a bash script ‚Äî if it reads JSON, it works with Percept.
